NAME: Free Facebook Page Template
AUTHOR: TemplateMonster
URL: http://www.templatemonster.com/free-templates/free-facebook-page-template.php

INSTRUCTIONS:
1. http://www.facebook.com/pages/create.php
2. In the Search field type "Static FBML"
3. Add to my page -> Choose page
4. In page settings go to Apps -> FBML-FBML -> Go to app

5. paste the following code:

	<fb:swf 
	swfbgcolor="ffffff" 
	imgstyle="border-width:0px; border-color:white;" 
	swfsrc='http://osc4.template-help.com/_flash/main_cs4.swf?path=http://osc4.template-help.com/_flash/'
	imgsrc='http://osc4.template-help.com/_flash/images/splash.jpg'
	width='520' height='700'/>

swfsrc - path to a SWF file
imgsrc - path to a splash image
